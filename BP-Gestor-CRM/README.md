# BP-Gestor-CRM

Este repositorio contiene la estructura de un Gestor de CRM. Incluye:

- Código fuente en la carpeta `src/OeaMatriz`.
- Scripts de base de datos en `db/OEA_schema_and_procedures.sql`.
- Documentación y matriz de requisitos en `docs/`.

La rama principal (`main`) almacena el contenido inicial.
